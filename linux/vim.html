Vim安装

sudo apt install vim

Vim配置

在用户目录下建立一个.vimrc文件，在里面添加相应配置,或在此目录下新建.vim文件夹，把vimrc、主题、缩进、不同文件类型的对应配置、插件...所有的用户配置都放置在里面，方便备份。

具体的配置项查看vim中文手册

Vim基本操作

常用的三种模式：插入，编辑，命令行。
插入模式正常除了输入文本，还有一些Ctrl-a,Ctrl-r Ctrl-k ctrl-p之类特殊的命令。
编辑模式的精髓在于快速移动光标定位，多用才能熟练。
按:进入命令行,可以执行很多命令...
